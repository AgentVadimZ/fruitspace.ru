

# ⌛ Логи - где их найти и для чего они?
Если для вас важно следить за действиями ваших модераторов, или вам нужно отследить человека, который совершил
какое-то нехорошее действие, на помощь придут логи, в которые записываются все действия на вашем GDPS.

## Краткий туториал по логам
1. В панели вашего сервера перейдите во вкладку **Логи** (![inline](@/assets/docs/small_icons/panel_logs.png))
2. Для сортировки действий можно использовать фильтры (![inline](@/assets/docs/small_icons/panel_logsfilter.png))
![Фильтр по действиям](@/assets/docs/images/logs_filter.png)
3. В списке действий отражена вся информация о каждом действии. Слева показана иконка события, большим текстом
сверху тип события, снизу дата события и его ID, а справа - игрок, который совершил событие.
![Пример списка действий](@/assets/docs/images/logs_overview.png)
